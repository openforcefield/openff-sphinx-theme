MyST

MyST MarkDown has some extra tricks up its sleeve. This theme is designed to be used with the following MyST extensions:

myst_enable_extensions = [
    "dollarmath",
    "colon_fence",
    "smartquotes",
    "replacements",
    "deflist",
]
